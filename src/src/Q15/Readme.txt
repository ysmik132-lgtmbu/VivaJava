This program demonstrates inheritance, polymorphism, and file handling.
The Employee class is the parent class, while FullTime and Contractor 
are subclasses that override the calculatePay() method. Employee objects
are stored in an ArrayList, and their salary details are written into a 
file named payroll_report.txt using BufferedWriter. This shows how Java 
can be used to generate structured reports using OOP and file handling.
